Save parameters in excel file: 'components.xlsx as shown in given template.
Following parameters should be chosen for turbine model generation.
B1 - Turbine "Tub" ID
B2 - Heat exchanger "HE" pipe length
B3 - Number of tubes in evaporator "Evap"
B4 - Number of tubes in superheater "Super"
B5 - Compressor ID "Comp"
B6 - Fuel pump ID
